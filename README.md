## Teste prático para Code Group (:

* Passos para preparar o ambiente
- Crie um banco de dados e configure com os dados inerentes a ele no .env
- Abra o terminal na raiz do projeto e digite o comando > $ composer install
- No terminal ainda execute o o comando > $ php artisan migrate 
- Execute o comando para ver se a migrations foram concluídas com sucesso > $ php artisan migrate:status
- Para subir o servidor excecute o comando > $ php artisan serve
